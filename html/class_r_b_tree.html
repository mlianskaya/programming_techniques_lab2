◆ insert()

void RBTree::insert ( const Applicant & a)

Вставка нового абитуриента в дерево.

Parameters
aОбъект Applicant.

Сначала выполняется обычный поиск места для вставки. Если ключ (ФИО) уже существует, объект добавляется в вектор values существующего узла. Иначе создается новый красный узел, который затем балансируется вызовом fixInsert().
