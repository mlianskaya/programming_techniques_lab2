\doxysubsection{Detailed Description}
Класс, описывающий абитуриента. 

Поля\+: ФИО, факультет, специальность, сумма баллов. Сравнение выполняется по правилам варианта 16\+:
\begin{DoxyItemize}
\item сначала сумма баллов по убыванию,
\item затем ФИО по возрастанию,
\item затем факультет по возрастанию,
\item затем специальность по возрастанию. 
\end{DoxyItemize}
