\doxysubsection{Detailed Description}
Заголовочный файл хэш-\/таблицы для хранения абитуриентов. 

Реализует хэш-\/таблицу с методом цепочек, где каждая корзина представляет собой отображение (std\+::unordered\+\_\+map) из ключа (ФИО) в вектор значений. Это позволяет эффективно хранить дубликаты и избегать линейного перебора при поиске.
