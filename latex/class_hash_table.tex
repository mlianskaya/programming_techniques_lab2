\doxysubsection{Detailed Description}
Класс хэш-\/таблицы для быстрого поиска по ключу. 


\begin{DoxyItemize}
\item Размер таблицы фиксирован и задаётся в конструкторе (рекомендуется простое число).
\item Коллизии разрешаются методом цепочек\+: каждая корзина — это std\+::unordered\+\_\+map.
\item Внутри корзины ключи уникальны, а для каждого ключа хранится вектор всех значений.
\item Дубликаты (одинаковые ФИО) не создают коллизий, а добавляются в существующий вектор.
\item Ведётся счётчик коллизий (сколько раз при вставке нового уникального ключа корзина уже была непуста). 
\end{DoxyItemize}
